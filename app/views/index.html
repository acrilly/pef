PEF Online prototype

This prototype has been built using the GOV.UK prototype kit, which lets you rapidly create HTML prototypes of GOV.UK services.

It contains Party Election Finance (PEF) prototypes to register Regulated Entities online.

Prototype Applications

  • Register a political party in Great Britain
  • Register as a non-party campaigner (third party) in Great Britain
  • Register a campaigner in a referendum
